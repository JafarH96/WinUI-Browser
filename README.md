# WinUI3 Browser

A modern web browser built with WinUI 3 and Microsoft Edge WebView2, offering a clean and intuitive user interface with essential browsing features.

## Technical Details

- **Framework**: .Net 8, WinUI 3
- **Language**: C# 13.0
- **Target Platform**: Windows 10/11
